nilmlite

A lightweight NILM data layer — explore datasets & run disaggregation models in your browser. pip install nilmlite

What is this? NILM (non-intrusive load monitoring), or energy disaggregation, recovers each appliance's power from a single whole-home smart meter — no per-plug sensors. Useful for energy feedback, demand response, and fault detection.

The hard part is deployment. A model trained in one home — or one country — has to work in homes it never saw. This page lets you watch that succeed and fail, live.

How it runs: models are trained in PyTorch on one dataset, exported to ONNX, and executed entirely in your browser via onnxruntime-web — no server, no upload. Data is stored as NILM-Parquet and loaded with nilmlite (pip-only, no Docker). Real datasets: REDD (US), UK-DALE (UK), iAWE (India).

3 · Cross-dataset generalization matrix (fridge)

Each model is trained on one dataset (row) and tested on every dataset (column), MAE in watts. Lower = better, colour-scaled green→red. The diagonal is within-dataset; the off-diagonal is cross-dataset — train in one country, test in another. That's where models break.
