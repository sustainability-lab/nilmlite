nilmlite

A lightweight NILM data layer — explore datasets & run disaggregation models in your browser. pip install nilmlite

What is this? NILM (non-intrusive load monitoring), or energy disaggregation, recovers each appliance's power from a single whole-home smart meter — no per-plug sensors.

The hard part is deployment. A model trained in one home — or one country — must work in homes it never saw. This page lets you watch that succeed and fail, live.

How it runs: models trained in PyTorch, exported to ONNX, executed entirely in your browser (onnxruntime-web) — no server. Data is NILM-Parquet loaded with nilmlite (pip-only, no Docker). Real datasets: REDD (US), UK-DALE (UK), iAWE (India).
