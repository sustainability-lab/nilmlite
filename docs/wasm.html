Why it matters

NILMTK/-contrib can't run in a browser at all. A reviewer (or a student, or a utility) can verify and run the NILM compute core from a URL — the benchmark becomes zero-install verifiable. Algorithmic kernels like CO even beat vectorized NumPy; SIMD reductions match it — but the point is portability at native-class speed, not a microbenchmark.
