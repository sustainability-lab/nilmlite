nilmlite studio

No-code disaggregation pipelines — load data, run a pretrained model, train one with real PyTorch, compare. ← explorer

How it works. Drag nodes (or load a sample), then ▶ Run. Pretrained models run in your browser via onnxruntime-web (PyTorch→ONNX). The Train node trains the real nilmlite PyTorch models (Seq2Point / DAE / GRU / PatchTST) on CPU or CUDA — via a local server you start with: pip install "nilmlite[dl]" && nilmlite serve, then open http://localhost:8000/. The graph serialises to JSON = a reproducible pipeline.
